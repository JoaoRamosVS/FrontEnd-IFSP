Programação Front-end

Programador

  • Implementa páginas web por meio de linguagens de programação como HTML, CSS e JavaScript
  • Trabalham com o design e as perspectivas do site
  • Responsável por colocar em prática, através de códigos, o design de um site
  • Garante que o site funcione corretamente e siga um design padrão em todos os dispositivos

Conhecimentos

  • HTML(HyperText Markup Language): é a linguagem utilizada para criar a estrutura e a semântica de um site ou aplicativo web
  • CSS (Cascading Style Sheet): é a linguagem utilizada para definir o estilo, a aparência e o layout dos elementos de um site ou aplicativo
  • JavaScript: é a linguagem de programação usada para adicionar interatividade e funcionalidades a um site
  • Design Responsivo: permite que a interface se adapte e seja exibida corretamente em diferentes dispositivos e telas
