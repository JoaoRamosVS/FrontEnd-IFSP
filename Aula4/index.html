CSS

Definição
Linguagem que especifica como os elementos HTML devem ser especializados em termos de cores, fontes, tamanhos, posição, margens, preenchimento, etc.

Origem
O CSS foi desenvolvido em 1996, por um grupo dentro do W3C (World Wide Web Consortium), para complementar o HTML.

Função
Separar o conteúdo do site de sua apresentação visual, alterando todo o aspecto estético da página.

Sintaxe
Utiliza regras (seletores) que especificam grupos de estilos que devem ser aplicados em elementos da página.
